# Three.js Web Project

Repo Template: three, eslint

## Building and Running on localhost

First install dependencies:

```sh
npm install
```

## Linting

To run static code analysis:

```sh
npm run lint
```

## Testing

To run unit tests:

```sh
npm run test
```
